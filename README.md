# React + TypeScript + Vite

## 🚀 Project Overview
This project is built using **React, TypeScript, and Vite**, offering a fast and efficient development experience. It serves as the main framework for the **Vistara** website.

## 📂 Project Structure
```
src/
├── pages/
│   ├── Portfolio.tsx    # Placeholder for Portfolio page
│   ├── Lequilibre.tsx   # L'Équilibre page
├── App.tsx              # Main app with routing
├── App.css              # Global styles
├── main.tsx             # Entry point
└── index.html           # Root HTML file
```

## 🛠️ Technologies Used
- **React** ⚛️ - Component-based UI
- **TypeScript** 📝 - Strongly typed JavaScript
- **Vite** ⚡ - Fast development & build tool
- **Framer Motion** 🎞️ - Animations and smooth UI transitions
- **React Router** 🔀 - Page navigation
- **Tailwind CSS** 🎨 - Utility-first styling

## 🚀 Getting Started
### **1. Clone the Repository**
```sh
git clone https://github.com/your-repo/vistara.git
cd vistara
```

### **2. Install Dependencies**
```sh
npm install
```

### **3. Run the Development Server**
```sh
npm run dev
```

### **4. Build for Production**
```sh
npm run build
```

## ✨ Features
- Interactive landing page with animated navigation
- Fully responsive design
- Dark-themed UI
- Navigation to "Portfolio" (future project) and "L'Équilibre" (Scale in French)
